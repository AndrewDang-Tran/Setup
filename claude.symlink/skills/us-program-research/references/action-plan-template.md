## Table Formatting Policy

ALL tables in the generated document MUST follow these rules:

```
1. Use Markdown pipe tables with proper alignment markers
2. Include header separators: :--- (left), :---: (center), ---: (right)
3. Align pipes vertically in source code for readability

Standard emoji usage:
  🥇🥈🥉  = Top 3 programs
  ⭐       = Recommended / highlighted
  ✅       = Confirmed / yes / available
  ❌       = Not available / no
  ⚠️       = Caution / warning
  🔴🟡🟢  = Status (not started / in progress / completed)
  ⛔       = Blocked
  💎       = Hidden gem
  🎖️       = Elite tier
  🔵       = Elective / secondary

Correct example:
  |  Rank | Program            | Cost  | Score |
  |:-----:|:-------------------|:-----:|:-----:|
  | **1** | **UT Austin MSAI** | $10k  |  89   |
  | **2** | **Rice MDS BA**    | $51k  |  87   |

Wrong example (do NOT do this):
  | # | Program | Cost | Score |
  |:-:|:--------|:----:|:-----:|
  | 1 | UT Austin MSAI | $10k | 89 |
```
